# servlet-feedback-form-app
This project is inspired from Durgesh Tiwari's second video on Servlet Crash Course.

[Servlet for Spring Boot | Servlet Crash Course | Servlet](https://youtu.be/N_meCl-5yEs?si=axKLlWXSBlmcempL)
Agenda:

1. State - session management
2. Develop a feedback form
3. Save user data in the database
4. State management by session and cookie
5. Connects to MySQL DB and saves the feedback
6. Dockerfile to deploy
7. docker-compose.yaml file to auto-deploy in a docker container.
